Работа с камерой и звуком

Цель задиния: Работа с мультимедиа. Создание мультимедийного приложения, позволяющего осуществлять работу с камерой и звуком(запись, воспроизведение аудио/видео файлов).Отображение полученных данных с использованием OpenGL.

AndroidManifest.xml

Файл манифеста предоставляет основную информацию о программе системы. Файл манифеста инкапсулирует всю архитектуру Android-приложения, eго функциональные возможности и конфигурацию. В процессе разработки вам придется постоянно редактировать данный файл, изменяя его структуру и дополняя новыми элементами и атрибутами.

Hеотьемлемым атрибутом нашего приложения является камера, мы будем использовать её для записи видео!

Для добавления возможности снимать видео при помощи нащего приложения нам нужно добавить тег uses-feature в AndroidManifest.xml.

После добавления тега для взаимодействия с камерой, мы переходим к созданию пользовательского интерфейса приложения.

Мы описываем две кнопки взаимодействия с камерой , а именно кнопку захвата видео и воспроизведения видео.

После этого мы должны имплементировать эти два метода в класс MainActivity.

Описание activity_video_play.xml.

Следующим действием мы должны создать Activity для воспроизведения видео VideoPlayActivity.

Результат нашей работы выглядит следующим образом.

В приложении предусмотрен диктофон. Диктофоном мы пользуемся, когда хотим записать важную для нас информацию в аудио формате. Перейдем к созданию пользовательского интерфейса для работы с диктофоном.

В AndroidManifest добавляем разрешение на запись звука.

Интерфейс насчитывает 4 кнопки для работы с диктофоном(начало записи, конец записи, воспроизведение записи и остановка воспроизведения записи).

Методы получения запроса на разрешение использования RECORD_AUDIO и рузультат выполнения запроса.

Настройка MediaRecorder.

Методом onCreate мы делаем проверку есть ли у нашего приложения доступ к микрофону.

Обработка событий каждого нажатия кнопок.

При нажатии копки Record начинается запись звука, длится до нажатия кнопки StopRecord. После сохранения записи можно прослушать её, нажав кнопку Play. Для приостановления прослушивания следует нажать кнопку StopPlay.

Результат проделанной работы:

В ходе выполнения лабороторной работы было создано приложение для работы с видео и аудио. Приложение способно осуществлять запись видео и аудио и дальнейшего его сохраниение, весь функционал осуществляется через удобный интерфейс.
